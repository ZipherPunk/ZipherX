Safety Constraints

NEVER Kill any processes — this crashes the app
ALWAYS Test TX building after ANY change to lib.rs
NEVER Change passwords without authorization
ALWAYS Test DB reads after ANY WalletDatabase.swift change
NEVER Run kill commands
ALWAYS Verify Secure Enclave after SecureKeyStorage changes
