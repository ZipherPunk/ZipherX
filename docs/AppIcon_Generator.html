ZIPHERX CYPHERPUNK ICON

Flash Green Z on Black - Pure Cypherpunk Ethos

To create the 1024x1024 icon:

  1. In Safari: Develop menu > Show Web Inspector
  2. Right-click the icon > "Capture Screenshot"
  3. Or use macOS screenshot (Cmd+Shift+4) on the icon
  4. Save as AppIcon-1024.png
  5. Place in Assets.xcassets/AppIcon.appiconset/
Cypherpunk Design:
- Pure black background (#000000)
- Neon green Z (#00FF00)
- Multiple glow layers for "flash" effect
- Subtle scanlines for retro-tech feel
