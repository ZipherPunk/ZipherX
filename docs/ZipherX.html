What is ZipherX?

A fully shielded cryptocurrency wallet for iOS. Zero trust. Zero servers. Zero knowledge of your transactions. Your keys. Your coins. Your privacy.

ZipherX is built on Zclassic ‚ÄĒ a community-driven cryptocurrency that forked from Zcash to remove the founders' reward and embrace true decentralization. Using cutting-edge zk-SNARKs cryptography, ZipherX enables fully private transactions where the sender, receiver, and amount are all encrypted on the blockchain.

Core Principles

  • Trustless by Design
    No central servers. No API dependencies. Direct P2P communication with the Zclassic network.
  • Multi-Peer Consensus
    Every piece of data is verified by multiple independent peers. A single malicious node cannot deceive your wallet.
  • Shielded Only
    ZipherX only supports z-addresses (shielded). Transparent addresses are not allowed. Privacy is mandatory.
  • Self-Custodial
    Your 24-word seed phrase is the only backup. We never see it. We can't recover it. You are sovereign.
  • Local Proof Verification
    All Sapling zk-SNARK proofs are generated and verified locally on your device.

Security Model

  • Keys Never Leave Device
    Your spending key is encrypted and stored locally. It never touches the network.
  • Proofs Generated Locally
    Sapling spend and output proofs are computed on-device using bundled parameters.
  • Header Verification
    Block headers are verified for proof-of-work. Invalid chains are rejected.
  • Nullifier Tracking
    Double-spend detection via nullifier set. Each note can only be spent once.
